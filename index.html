Box Model in CSS

What is Box Model?

Everything in a webpage is a rectangular box. The text, buttons, inputs, pictures all of them are just a box, also a circle drawn using CSS will be represented as a box when it is drawn to screen. And this box is constitute of content, padding, border and margin which in total makes up box model.

Content

Whatever is inside the box is called the content of box. This content can be text, image or even just fixed width and height.

Padding

Padding is exactly outside the content and it controls the space between border of the box and the content of the box.

Border

Border is space between the margin and the padding of the box.

Margin

Margin is space between the border of box itself and borders of adjacent boxes.

Margin Collapse

If two html elements are sibling to each other, and they both have a margin then their margin will collapse so only the largest of two margin is used between the elements.

Box Sizing

box-sizing property can be used to change how the box model calculates sizes and the default value of it is content-box, which means the content of the element will be the exact width/height defined by the width and height properties in CSS. This value can be changed to border-box which means the width/height properties in CSS will represent the width/height of the content combined with the border and padding.
